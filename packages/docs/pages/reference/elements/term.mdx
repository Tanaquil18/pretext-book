# `<term>`

The `<term>` tag should be used when defining a word or phrase.  

## Syntax

<SyntaxDisplay name="term" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    The <term>binomial coefficient</term>, denoted <m>\binom{n}{k}</m>,

```
