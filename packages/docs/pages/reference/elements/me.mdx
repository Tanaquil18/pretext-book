# `<me>`

Expressions enclosed by the `<me>` tag are rendered as a single line of displayed mathematics. By contrast, the [`<m>`](./m) tag is used for in-line mathematics.  

## Syntax

### Attributes

<AttrDisplay name="me" />

### Children

The following can be used as children.

<ChildrenDisplay name="me" />

### Child of

This element can appear as an immediate child of the following elements. If the equation concludes a sentence, put the concluding punctuation after the closing tag.

<ParentsDisplay name="me" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    <me>\int \cos x\, dx=\sin x+C</me>
</FRAGMENT>
```
