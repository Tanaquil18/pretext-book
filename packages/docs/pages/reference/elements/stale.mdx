# `<stale>`

The `<stale>` tag marks it's contents as _stale_ to the reader by rendering it
in <span style={{ textDecoration: "line-through" }}>strike-through text</span>.

## Syntax

<SyntaxDisplay name="stale" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    When proofreading, you may <stale>mark stale</stale> parts of your work.
</FRAGMENT>
```
