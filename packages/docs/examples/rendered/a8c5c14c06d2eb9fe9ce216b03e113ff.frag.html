Subsection 1.1

Principle 1.1.1 (Creator Optional). The Prinicple of Mathematical Induction.

To show that a mathematical statement, \(P(n)\text{,}\) indexed by integers is true for all integers \(n \ge a\text{,}\) one need only show
  1. P(a) is true, and
  2. for any \(k \ge a\text{,}\) whenever \(P(k)\) is true, so is \(P(k+1)\text{.}\)
