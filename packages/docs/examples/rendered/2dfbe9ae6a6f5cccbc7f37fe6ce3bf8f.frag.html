Subsection 1.1

Insight 1.1.1. Pivot columns.

When we consider a linear system of the form \(Ax=b\text{,}\) we know (via Gaussian elimination) that the system is solvable if and only if the rightmost column of the augmented matrix \([A|b]\) is not a pivot column, which is to say that the reduced row-echelon form of the augmented matrix does not have a row of the form \([0\ 0 \dots 0\ *]\) where \(*\) is a nonzero scalar.
