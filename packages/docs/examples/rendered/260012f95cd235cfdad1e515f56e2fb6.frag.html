Subsection 1.1

Observation 1.1.1. A very useful observation.

If a matrix is given by \(A= [a_1 a_2 \dots a_n]\) where \(a_j\) is the \(j\)th column of \(A\text{,}\) and if \(x\) is the column vector \(x =[x_1, \dots, x_n]\text{,}\) then the matrix product \(Ax\) is a linear combination of the columns of \(A\text{.}\) More precisely,
\begin{gather*} Ax = x_1 a_1 + \cdots + x_n a_n. \end{gather*}
