Subsection 1.1

Example 1.1.1. Testing vector space axioms.

Polynomials of degree 3 do not form a vector space.

Answer.

The set is not closed under addition and lacks an additive identity.

Solution 1.

The sum of elements \(x^3 + x^2\) and \(-x^3 + x\) equals \(x^2 + x\) which is not of degree 3, so the set is not closed under vector addition.

Solution 2.

We can also note that the zero polynomial does not have degree 3, so is not an element of the set.
