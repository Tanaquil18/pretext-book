Subsection 1.1

Remark 1.1.1. Eigenvectors and kernels.

Given a linear map \(T:V \to V\) on a vector space \(V\) over a field \(F\text{,}\) an eigenvalue is a scalar \(\lambda\in F\) for which there exists a nonzero vector \(v\in V\) (an eigenvector) for which \(T(v) = \lambda v\text{.}\) At first blush this may seem an odd notion, but it becomes less so when we observe that eigenvalues are defined to be those scalars \(\lambda\in F\) for which \(\ker(T-\lambda I_V) \ne 0.\)
