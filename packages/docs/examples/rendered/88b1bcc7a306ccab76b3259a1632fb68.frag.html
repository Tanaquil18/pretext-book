Subsection 1.1

Note 1.1.1. Maps out of quotients.

In almost any algebraic context, you should never try to a define a map directly whose domain is a quotient, rather try to find a natural way to invoke the first isomorphism theorem.
For example if \(V,W\) are vector spaces and \(U\) is a subspace of \(V\text{,}\) rather than try to define a map \(T:V/U \to W\text{,}\) try instead to define a map \(T_0:V\to W\) whose kernel is \(U\) and image is the same as that of \(T\text{.}\) It is easier than you think and always well-defined.
