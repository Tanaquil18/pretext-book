Subsection 1.1

Convention 1.1.1. Optional.

For an arbitrary group \(G\text{,}\) we typically denote the operations multiplicatively, so for elements \(a,b \in G\text{,}\) we would write \(a\cdot b\) or \(a^{-1}\) for the the application of the group operation or the inverse of an element. However, when a group is known to be abelian, the notation is often written additively, such as \(a+b\) or \(-a\text{.}\)
